##### Sinatra boilerplate

The complete sinatra (*mvc) boilerplate ready for building simple applications

##### Requirements

ruby 2.4.1

##### Components

- views: Haml
- authentication: Warden
- orm: Sequel
- database: SQLite
- testing:
  - RSpec
  - FactoryGirl
- styling: Bootstrap
- bundler

##### Development

shotgun config.ru
